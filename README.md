- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/zighunter.git
cd zighunter
go build -o zighunter .
./zighunter --help
```

## 🚀 Quick Start

```bash
./zighunter -h
```

## 🧪 Examples

```bash
# Example 1
./zighunter status

# Example 2
```
