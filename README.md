# ZIGHUNTER

**High-Performance HTTP Security Scanner**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/zighunter.git
cd zighunter
go build -o zighunter .
```

## Usage

```bash
./zighunter --help
```
